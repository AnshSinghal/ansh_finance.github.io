Local Development

Run Locally

Use these commands to run the Django API on your machine.

Setup Commands

python -m venv .venv
source .venv/bin/activate
pip install -e ".[test]"
cp .env.example .env
python manage.py migrate
python manage.py runserver

Validation

Run tests with:

pytest tests/ -v

Docker Option

docker compose up --build
