Architecture Boundary

Pages and API Deployment Model

GitHub Pages serves static docs; Django serves runtime API traffic.

What GitHub Pages Hosts

Only static files from the repository, such as HTML, CSS, images, and markdown.

What GitHub Pages Does Not Host

Django app runtime, database, Celery jobs, or dynamic API execution.

Recommended Runtime Targets

Deploy Django backend on a runtime platform (for example Render, Railway, Fly.io, ECS, or a VM).
