Endpoint Map

API Reference Snapshot

Base path: /api/v1. Use the runtime Swagger UI for full request/response schemas.

Group Endpoints Access
Auth POST /auth/register/, POST /auth/login/, POST /auth/logout/, POST /auth/refresh/ Public for register/login
Users GET /users/, GET /users/me/, PATCH /users/{id}/role/ Admin for management endpoints
Records GET /records/, POST /records/, PATCH /records/{id}/ Role-restricted (admin for write)
Dashboard GET /dashboard/summary/, /categories/, /trends/, /recent/ Analyst/admin for categories and trends

Runtime Interactive Docs

When the backend is running locally, use: http://127.0.0.1:8000/docs/ and http://127.0.0.1:8000/redoc/.
